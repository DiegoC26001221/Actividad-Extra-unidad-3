Esta tercera unidad se trato meramente de aprender a usar la nueva herramienta llamada git junto con la pagina web git hub, aprendimos a como agregar un archivo a la rama principal, a como agregar branchs, o raíces  para asi  poder realizar  funciones distintas y también a como editar el texto  mediante comandos. De toda esta unidad yo destaco por sobre todas las cosas los comandos que utilizamos, pues no fueron muchos y por ende, se me lograron quedar mas claros a la hora de que hace cada uno, como el git init, git add . , el git commit etc, fue muy gratificante y divertido emplear cada código en las distintas carpetas y luego ver  como cambiaba el repositorio en tiempo real.
